たとえると... 料理人が新しい調理技術を習得するイメージ。AI がスキルを持つことで、その分野のプロ並みの仕事ができるようになります。

browser-presentation スキルとは

ブラウザで動くプレゼンスライドを自動生成するスキルです

  • テーマや内容を伝えるだけで OK
  • HTML / CSS / JavaScript は自動で生成される
  • デザインセンスも不要 — プロ品質に仕上がる
  • 完成ファイルをブラウザで開けばすぐ発表できる
💡 このスライド自体も browser-presentation スキルで作られています！

スキルの設定方法

お使いの AI ツールのスキルディレクトリへ Git でクローン、または ZIP で配置します。

Claude Code
グローバル設定（全プロジェクト） git clone https://<URL>/browser-presentation-public.git ~/.claude/skills/browser-presentation プロジェクト設定（現在のプロジェクトのみ） git clone https://<URL>/browser-presentation-public.git ./.claude/skills/browser-presentation
IBM Bob
グローバル設定（全プロジェクト） git clone https://<URL>/browser-presentation-public.git ~/.bob/skills/browser-presentation プロジェクト設定（現在のプロジェクトのみ） git clone https://<URL>/browser-presentation-public.git ./.bob/skills/browser-presentation
⚠️ 重要: フォルダ名は必ず browser-presentation にしてください（エクスポートスクリプト内のパスと一致させるため）。
💡 Gitが使えない場合: リポジトリから ZIPでダウンロード して解凍し、フォルダ名を変更して配置することも可能です。

Claude での使い方

1
Claude Code を開く

ターミナルまたは Claude アプリを起動する

2
スキルを呼び出す

チャットに /browser-presentation と入力する

3
内容を伝える

「〇〇についてのスライドを作って」と依頼する

4
index.html をブラウザで開く

生成された 3 ファイルの index.html を開けば完成！

IBM Bob での使い方

1
IBM Bob を開く

IBM Bob チャット画面にアクセスする

2
スキルを呼び出す

入力欄に "browser-presentation"を使用して というようにスキル使用を明示する

3
内容を伝える

「〇〇についてのスライドを作って」と依頼する

4
ファイルを受け取って開く

生成されたファイルをダウンロードし、index.html を開く

依頼例 — 基本の書き方

テーマ・対象者・枚数・デザインを伝えるだけで OK

シンプル
「新入社員向けのセキュリティ研修スライドを作って」
詳細指定
「〇〇プロジェクトの進捗報告スライドを 5 枚で作って。
対象は経営層向け。ダークテーマで」
スタイル指定
「AI 活用推進の社内提案スライドを作って。
明るく親しみやすいデザインで、図解を多めに」
💡 コツ：「誰に向けて」「何枚」「どんなデザイン」を伝えると、よりイメージ通りに仕上がります

依頼例 — 他の資料を参照する

既存の文書・メモ・URLを渡してスライド化できる

ファイル添付
「添付の会議メモを読んで、議事録スライドを作って」
→ Word / PDF / テキストファイルを一緒に渡す
テキスト貼付
「以下の製品仕様書をもとに、顧客向け提案スライドを作って：
（仕様書の内容をここにコピー&ペースト）」
既存資料を元に
「去年のスライド構成を参考にしながら、
今年版の方針発表スライドを作って」
→ 既存スライドのテキストや構成を貼り付けて渡す
💡 ポイント：「何を参考にしてほしいか」を明示するほど、内容の精度が上がります

PPTX・PDF に変換するには

🗣️ 方法① AI に依頼する（一番かんたん） Claude・IBM Bob 両対応
「PPTX に変換して」
「PDF に変換して」

スライド生成後にそのまま続けて依頼するだけ

🖨️ 方法② ブラウザから PDF 保存 ブラウザのみ
  1. index.html をブラウザで開く
  2. Cmd+P（Mac）/ Ctrl+P（Windows）
  3. 「PDF として保存」を選択
  4. 各スライドが 1 ページずつ出力される
⌨️ 方法③ コマンドで手動変換 Claude・IBM Bob 両対応
Claude Code
node ~/.claude/skills/browser-presentation/export-pptx.js index.html output.pptx node ~/.claude/skills/browser-presentation/export-pdf.js index.html output.pdf
IBM Bob
node ~/.bob/skills/browser-presentation/export-pptx.js index.html output.pptx node ~/.bob/skills/browser-presentation/export-pdf.js index.html output.pdf

※プロジェクト設定でインストールした場合は、先頭の ~/ を ./ に読み替えてください

エクスポートの事前準備（手動変換時）

コマンドで変換する場合、初回のみ以下の設定が必要です

1
必要なモジュールをインストール npm install -g @mermaid-js/mermaid-cli pip install python-pptx
2
スクリプト内の Puppeteer パスを修正

export-pptx.js と export-pdf.js の先頭行にある require() のパスを、ご自身の環境（Homebrew, Volta, nvm等）に合わせて修正してください。

💡 インストールや設定が不安な場合は、AIのサンドボックス環境を利用する「方法① AIに依頼する」がお手軽でおすすめです！
